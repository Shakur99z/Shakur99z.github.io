// How Entropy Is Calculated

Password entropy measures unpredictability. It's calculated as E = L × log₂(P), where L is password length and P is the size of the character pool used (lowercase: 26, uppercase: 26, digits: 10, symbols: 32).

A password with 80+ bits of entropy is considered strong against brute force. The crack time estimate assumes an attacker can make 1 billion guesses per second — realistic with modern GPU rigs. A truly random 16-character password with mixed characters would take trillions of years to crack.
